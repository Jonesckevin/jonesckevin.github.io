# Competency Standards — Brigadier-General / Commodore (BGen/Cmdre)

> **2026 Competency Framework Reference**
> Use this document when scoring Behavioural Indicators (BIs) in PaCE for a BGen/Cmdre.
> **Score 3 = Effective = the expected standard.**
> For next-rank indicators, refer to [mgen.md](mgen.md).

---

## Rating Scale Quick Reference

| Score | Label | Meaning |
|---|---|---|
| 1 | Ineffective | Does not demonstrate this behaviour. Requires immediate intervention. |
| 2 | Partially Effective | Demonstrates inconsistently. Needs coaching and follow-up. |
| 3 | Effective | Consistently meets the BI as written. The expected standard. |
| 4 | Above Effective | Consistently meets BI; occasionally shows MGen-level behaviours. |
| 5 | Extremely Effective | Routinely demonstrates MGen-level behaviours. Reserve for outstanding performance. |
