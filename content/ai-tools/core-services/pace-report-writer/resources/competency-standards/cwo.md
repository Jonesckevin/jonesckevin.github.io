# Competency Standards — Chief Warrant Officer / Chief Petty Officer 1st Class (CWO/CPO1)

> **2026 Competency Framework Reference**
> Use this document when scoring Behavioural Indicators (BIs) in PaCE for a CWO/CPO1.
> **Score 3 = Effective = the expected standard.**
> Score 5 behaviours reflect MGen / institutional-level standards where applicable; otherwise Col/BGen-level.

---

## Rating Scale Quick Reference

| Score | Label | Meaning |
|---|---|---|
| 1 | Ineffective | Does not demonstrate this behaviour. Requires immediate intervention. |
| 2 | Partially Effective | Demonstrates inconsistently. Needs coaching and follow-up. |
| 3 | Effective | Consistently meets the BI as written. The expected standard. |
| 4 | Above Effective | Consistently meets BI; occasionally shows institutional-level behaviours. |
| 5 | Extremely Effective | Routinely demonstrates institutional/MGen-level behaviours. Reserve for outstanding performance. |
